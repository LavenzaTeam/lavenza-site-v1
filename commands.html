Lavenza Commands

Note, this list is incomplete and is subject to change at any moment!




Command Name Description Usage
Help Displays all of the current commands for the bot! ;help (command)
Invite Sends a link to the current channel that you can use to invite Lavenza to any server! ;invite
Support Sends a link to the current channel that you can use to join the support server! ;support
Avatar Sends a picture of you or a specified user's avatar on Discord! ;avatar (@user)
Ping Gets the latency from Lavenza's servers and the latency from the Discord API! ;ping
Version Get's the current version of Lavenza and all of it's dependencies that it uses! ;version
P3 A command that tries to condense all of the other Persona 3 commands into one command. /a will give you the answer variant, /f will get you the FES Fusion of a Persona, and /p will give you the Portable Fusion of a Persona. Note that you may only append one switch at a time. ;p3 [persona/shadow] (/a) (/f) (/p)
